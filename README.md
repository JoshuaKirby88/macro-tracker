# Onigiri

AI-powered macro tracking app. Track daily nutrition with meal-based organization, progress tracking, and intelligent food management.

## Features

- **Food Tracking** — Log entries by meal (breakfast/lunch/dinner) with quantity management and real-time progress bars
- **Food Database** — Personal food library with search (Cmd+K), nutritional info, and image integration
- **Goals** — Set meal-specific targets for calories, protein, fat, carbs, and fiber
- **AI Analysis** — Scan nutrition labels via image upload to auto-extract macro data
- **Export** — Export entries as markdown with date range, meal, and nutrition field filtering

## Tech Stack

- Next.js / React / TypeScript
- Convex (backend/database)
- Clerk (auth)
- Gemini (AI image analysis)
- Cloudflare Workers (deployment)
